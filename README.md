# To-Do-app-python
To-Do App (Python CLI)

A simple command-line To-Do application built using Python.
This project allows users to add, view, and delete tasks during runtime using a menu-driven interface.

⸻

 Project Description

This To-Do App is a beginner-friendly Python project that demonstrates core programming concepts such as loops, lists, conditionals, and user input handling.
Tasks are stored temporarily in memory using a Python list and remain available until the program exits.

⸻

 Features
	•	 Add new tasks
	•	 View all tasks
	•	 Delete tasks by selecting task number
	•	 Continuous menu until user exits
	•	 Simple and fast command-line interface
  Tech Stack
	•	Language: Python 3
	•	Concepts Used:
	•	Lists
	•	While loop
	•	Conditional statements (if-elif-else)
	•	User input (input())
	•	Basic indexing and list operations

How It Works
	1.	Program runs inside an infinite while loop
	2.	User selects an option from the menu
	3.	Tasks are stored in a Python list
	4.	User can add, view, or delete tasks
	5.	Program exits when user selects option 4
